# Missing  
**Call or text your lost phone. It replies with location — even on silent.**

<img src="https://raw.githubusercontent.com/bitebifusion/missing/main/assets/logo.png" width="200" align="right" />

The open-source “lost phone” app that actually works in 2025.  
One click to install → runs forever in background → zero battery drain → zero tracking.

## Features
- Works when phone is on silent/vibrate  
- Auto-reply with live Google Maps link  
- Custom message + reward  
- Password protected  
- Works offline after first setup  
- 100% free • 100% open-source • No ads ever
